# 🕌 بوت حصن المسلم

بوت تليجرام إسلامي شامل يحتوي على الأذكار والأدعية الثابتة من القرآن الكريم والسنة النبوية الصحيحة.

## ✨ المميزات

- 🌅 أذكار الصباح (19 ذكراً بأدلتها)
- 🌇 أذكار المساء (17 ذكراً بأدلتها)
- 🌙 أذكار النوم والاستيقاظ
- 🕌 أذكار بعد الصلاة
- ✈️ أذكار السفر والمناسبات
- 🛡️ الرقية الشرعية
- 🌸 أسماء الله الحسنى الـ99
- 💓 التسابيح والأدعية المأثورة
- 📖 حديث اليوم (30 حديثاً يتغيرون يومياً)
- 🎲 ذكر عشوائي
- 🔔 تذكير تلقائي ثلاث مرات يومياً حسب توقيت البلد:
  - الساعة 6:00 صباحاً ← أذكار الصباح
  - الساعة 8:00 صباحاً ← حديث اليوم
  - الساعة 7:00 مساءً ← أذكار المساء
- 🌍 دعم 31 دولة بمناطقها الزمنية الصحيحة
- تنقل بين الأذكار بأزرار التالي / السابق / الرئيسية

## 🛠️ متطلبات التشغيل

- Node.js 18 أو أحدث
- npm

## 🚀 التثبيت والتشغيل

### 1. انسخ المشروع

```bash
git clone <رابط مشروعك>
cd islamic-azkar-bot
```

### 2. ثبّت الحزم

```bash
npm install
```

### 3. إعداد المتغيرات البيئية

انسخ ملف `.env.example` إلى `.env`:

```bash
cp .env.example .env
```

ثم عدّل الملف بقيمك:

```
TELEGRAM_BOT_TOKEN=توكن_البوت_من_BotFather
ADMIN_ID=معرّفك_على_تليجرام
PORT=8080
```

### 4. بناء وتشغيل

```bash
npm run dev
```

## 🌐 النشر على Render

1. ارفع المشروع على GitHub
2. اذهب إلى [render.com](https://render.com) وأنشئ **Web Service** جديداً
3. اربطه بمستودع GitHub
4. في إعدادات الخدمة:
   - **Build Command:** `npm install && npm run build`
   - **Start Command:** `npm start`
5. في قسم **Environment Variables** أضف:
   - `TELEGRAM_BOT_TOKEN` ← توكن البوت
   - `ADMIN_ID` ← معرّفك على تليجرام
6. اضغط **Deploy** وانتظر

## 📁 هيكل الملفات

```

├── bot/
│   ├── index.ts       ← منطق البوت الرئيسي والأوامر

│   ├── hadith.ts      ← أحاديث اليوم
│   ├── keyboards.ts   ← تصميم الأزرار
│   ├── scheduler.ts   ← المجدول للتذكيرات اليومية
│   └── storage.ts     ← حفظ بيانات المستخدمين (JSON)
├── lib/
│   └── logger.ts      ← نظام السجلات
├── routes/
│   ├── index.ts       ← توجيه المسارات
│   └── health.ts      ← فحص صحة الخادم
├── app.ts             ← إعداد Express
└── index.ts           ← نقطة الدخول
```
